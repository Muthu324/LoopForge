# Runtime

## Purpose

The Runtime is the heart of LoopForge.

It coordinates every execution.

---

## Responsibilities

- Start execution
- Track state
- Collect traces
- Capture errors
- Record latency
- Record token usage
- Send events to storage

---

## Inputs

- Agent
- Prompt
- Configuration

---

## Outputs

- Execution Trace
- Metrics
- Errors
